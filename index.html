Accessible Airlines

Accessible Airlines is my personal project which is still on-going partly because my coding skills in PHP are not as good as they used to. Currently it allows browsing through several airlines. Future plans include comparison between airlines on one page.

The most complicated part of this project was choosing which airline to use as a base line because some airlines, e.g. Air France, have complex but clear document, others, e.g. Finnair, have information in several places.

Writerry

I created Writerry during my ongoing YouTube series - 12 Weeks design to code challenge. Main purpose of this website was testing CSS variables and showing that a designer/developer can make accessible websites. Main point is that developers should respect hierarchy of headings because multiple h1s and h2s may confuse a person who can't see the website itself and relies on a screen reader.
